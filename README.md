## O QUE É TRY-CATCH
 - Try-Catch é quando o codigo da um erro, ao contrario de aparecer varias mensagem "feias" o Try-Catch faz com que mostre uma mensagem amigavel, no exemplo do codigo aparece "[ERRO DE UX]: Você digitou letras em um campo que só aceita números!" e "DICA: Tente novamente inserindo apenas algarismos (ex: 25)."
